# 📊 Centro de Estudio — Probabilidad y Estadística (2do Parcial)

Kit de estudio interactivo para el **segundo parcial de Probabilidad y Estadística** (UTN FRC), enfocado en las **Unidades 6 a 9** y en un examen teórico-práctico.

Todo funciona como páginas web estáticas (HTML + CSS + JavaScript, sin dependencias ni servidor): se abren directamente en el navegador.

---

## 🧭 ¿Qué incluye la app principal?

La app (`index.html`) está organizada en pestañas:

- **🗓️ Plan 3 días** — Plan de estudio de 3 días (4 a 6 h/día) con estrategia 80/20, barra de progreso y casilleros para ir tildando cada bloque a medida que se completa.
- **U6 · Modelos continuos** — Modelos especiales de probabilidad con variable continua: **Normal, Exponencial, Uniforme y t de Student**, con árbol de decisión para elegir el modelo correcto.
- **U7 · Muestreo** — Teoría del muestreo, distribución de la media muestral y Teorema Central del Límite (TCL).
- **U8 · Estimación** — Estimación estadística: intervalos de confianza y tamaño de muestra.
- **U9 · Dócimas** — Contraste / verificación de hipótesis (pruebas de hipótesis).
- **🃏 Flashcards** — Tarjetas de recuerdo activo (con efecto de giro) para repasar conceptos y fórmulas clave.
- **✅ Quiz** — Cuestionario autoevaluable con corrección inmediata, explicaciones y puntaje.
- **📝 Simulacro** — Simulacro de examen con corrección paso a paso.
- **📋 Examen real** — Análisis del examen real del 2do parcial, con su resolución detallada usando InfoStat.
- **📐 Formulario** — Formulario maestro con todas las fórmulas en una sola pantalla.

> 💡 La práctica está pensada para usarse junto con **InfoStat**. Por eso se incluye la guía rápida del software como material complementario.

## 🚀 Cómo usarlo

No requiere instalación ni conexión a internet.

1. Cloná o descargá este repositorio.
2. Abrí **`index.html`** con un doble clic (o desde tu navegador).
3. Para la guía de InfoStat, abrí **`guia-infostat/index.html`**.
4. El **PDF del formulario** se puede imprimir o tener abierto mientras practicás.

```bash
git clone https://github.com/Santuriano/SegundoParcialPyE.git
cd SegundoParcialPyE
# Abrí index.html en tu navegador
```
