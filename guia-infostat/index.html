⭐ 4 reglas de oro (acordate de estas SÍ o SÍ)

1. Coma decimal, NO punto. Si escribís 0.125 InfoStat tira el error "Algún parámetro especificado no es numérico". Tenés que escribir 0,125. (Me pasó en vivo: con punto da error, con coma calcula.)
2. Para borrar un campo NO sirve Ctrl+A. Si tocás el campo y escribís, el número se concatena al que ya estaba (queda 1225 en vez de 225). Borralo con Backspace o seleccioná con el mouse antes de escribir.
3. La Normal pide VARIANZA (σ²), no la desviación. Si el problema dice σ=15, cargás 225. Si dice σ=40, cargás 1600.
4. La Exponencial pide LAMBDA, no la media. Si el problema da media μ, cargás lambda = 1/μ. Ej: μ=8 → lambda=0,125. ⚠️ El error más común del parcial.

📈 Normal — X ~ N(μ ; σ)

Parámetros a cargar: media = μ varianza = σ²

Te piden…Qué hacés en InfoStat
P(x ≤ k)  (o <)Valor de x = k → leés Prob. (X≤x)
P(x ≥ k)  (o >)Valor de x = k → leés Prob. (X>x)
P(k₁ ≤ x ≤ k₂)Calculás dos veces: P(x≤k₂) − P(x≤k₁) (ver sección "entre")
Ejemplo verificado: N(60;15) → media 60, varianza 15²=225, x=90, Calcular → P(X≤x)=0,9772 y P(X>x)=0,0228. ✔️
Para la media muestral (U7): es una Normal igual, pero cargás varianza = σ²/n. Ej: σ=40, n=120 → varianza = 1600/120 = 13,33.

⏳ Exponencial

Parámetro a cargar: lambda = 1/μ

Si el problema te da la media μ (lo habitual), NO la cargues directo. Convertí: lambda = 1/μ.
Ejemplo verificado: media μ=8 → lambda = 1/8 = 0,125. Valor de x=16, Calcular → P(X>x)=0,1353 = e⁻². ✔️ (idéntico al cálculo a mano).

Tip: para P(x>a) leés "Prob.(X>x)"; para P(x<a) leés "Prob.(X≤x)". Para P(a<x<b) restás como en la Normal.

📦 Uniforme (a,b)

Parámetros a cargar: a = mínimo b = máximo

Valor de x = el punto que te piden → leés Prob.(X≤x) o Prob.(X>x). Para "entre", restás.

La media E(x)=(a+b)/2 y la varianza V(x)=(b−a)²/12 conviene hacerlas a mano (son directas y a veces las piden aparte).

🔔 t de Student

Parámetro a cargar: v = grados de libertad = n − 1

Se usa con muestras chicas (n<30) y σ desconocida. Mismo mecanismo: Valor de x = el valor de t → leés la probabilidad. Y al revés con el problema inverso (abajo) para hallar el t crítico.

Del repaso: P(t < a)=0,90 con n=12 (v=11) → con el inverso obtenés a=1,3634.

🔁 Problema inverso (dan la probabilidad, piden el valor k)

Aparece siempre: "Hallar k tal que P(x≤k)=0,9845" o "la nota que supera al 31,5%". En InfoStat es facilísimo:

Cargás los parámetros (media, varianza, etc.) como siempre.
Dejás vacío el campo "Valor de x".
Escribís la probabilidad en el campo Prob. (X≤x) (o Prob. (X>x) si te la dan por derecha).
Apretás Calcular → te aparece el Valor de x = la k buscada.
Verificado: N(60;225), Prob.(X≤x)=0,9845, Valor de x vacío → Calcular → k = 92,36. ✔️ Exacto al repaso.

📐 Probabilidad "entre" dos valores

Opción A (la más segura): restar dos resultados.

P(k₁ ≤ x ≤ k₂) = P(x≤k₂) − P(x≤k₁)

Calculás P(x≤k₂), anotás; calculás P(x≤k₁), anotás; restás. Ej del repaso: P(75≤x≤90)=0,9772−0,8413=0,1359.

Opción B (gráfica): Aplicaciones → Didácticas → Gráficos de funciones de densidad continuas. Elegís la distribución, opción "entre", ingresás los dos límites. Para las dos colas (P fuera del intervalo) tildás "Complementario".

La opción B además te dibuja el área sombreada — útil si el examen muestra un gráfico y pide el área rayada.

🧪 Intervalos de confianza · Dócimas · Tamaño de muestra

Estas funciones sí necesitan una tabla de datos cargada (columna con los valores de la muestra). Primero: Archivo → Nueva tabla (o abrir el .xlsx/.IDB2) y pegás los datos en una columna.

Querés…Ruta en InfoStat
Intervalo de confianza (media / proporción)Estadísticas → Estimación de características poblacionales
Dócima / test de hipótesis (una muestra)Estadísticas → Inferencia basada en una muestra
Comparar dos gruposEstadísticas → Inferencia basada en dos muestras
Tamaño de muestraEstadísticas → Cálculo del tamaño muestral (no requiere datos)
Medidas resumen (media, desvío, etc.)Estadísticas → Medidas resumen
Potencia de la dócima (1−β): InfoStat calcula automáticamente el punto crítico (x̄*) según α. Para la potencia en un μ dado, "clonás" el gráfico e ingresás ese μ. Ej del material: n=50, σ²=102,42 → σ²(x̄)=2,0484; μ₀=136; con α=5% el punto crítico es x̄*=138,354; para μ=139,46 la potencia 1−β=0,7801.
Si en el examen te dejan elegir, para probabilidades de modelos (U6/U7) el calculador es lo más rápido; para IC y dócimas con datos, usás los menús de inferencia o resolvés a mano con el formulario.
